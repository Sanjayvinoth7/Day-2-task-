DIFFERENCE  BETWEEN  DOCUMENT  AND  WINDOWS  OBJECT  :


DOCUMENT :
                     
                                      1. The  document  object  represent  a  web  page  that  is  loaded  in  the  browser. By  accessing  the  document  
object , we  can  access  the  element   in   the   HTML  pages.


                                      2. With  the  help  of  document  objects,  we  can  add  dynamic  content  to  our  web  page. The  document  object
can  be  accessed  with  a  Window.document  or  just  document.


                                      3. It  represents  any  HTML  document   or  web  page  that   is  loaded  in  the  browser. It  is  loaded  inside  the 
window. And  it  is  the  object  of  window  property.

                            
                                      4. Properties  of  document  object  such  as  title, body , cookies , etc  can  also  be accessed  by  a  window  like
this  window.document.title .



WINDOWS :
 
                                     1. The  window  object  is  the  topmost  object  of   the  DOM  hierrachy. It  represents  a  browser  window  or  frame
that  displays  the  contents  of  the  webpage.


                                     2. Whenever  a  window  appears   on  the  screen  to  display  the  contents  of  the  document , the  window  object  
is  created.


                                     3. It  represents  a  browser  window  or  frame  that  displays   the   content   of  the  webpage. It  is  the  very  first
object   that   is   loaded   in  the  browser.


                                     4.  Properties  of  the  window  object  cannot  be  accessed  by  the  document  object.
